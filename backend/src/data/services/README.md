# كتالوج الخدمات (Services Catalog)

ضع هنا ملفات الخدمات بصيغة **CSV** (أجزاء سلة تيبس):

```
خدمات_جزء_1.csv
خدمات_جزء_2.csv
...
خدمات_جزء_10.csv
```

## الأعمدة المطلوبة (الصف الأول = العناوين)

```
id,name,type,status,price,sale_price,currency,category,url,sold_quantity,rating,problems_solved,description
```

أهم عمود للترشيح هو **`problems_solved`** — قائمة المشاكل التي تحلها الخدمة، مفصولة بـ ` | `.

## ملاحظات

- الملفات تُحمّل تلقائياً في الذاكرة عند أول استخدام، وتُعاد قراءتها تلقائياً عند تغيّر أي ملف (لا حاجة لإعادة تشغيل السيرفر).
- يجب أن تكون الملفات بترميز **UTF-8** (مع أو بدون BOM).
- الصفوف المكررة (حسب `id`) تُتجاهل تلقائياً.
- لا تُرفع هذه الملفات للذكاء الاصطناعي كاملة — يتم فقط ترشيح الخدمات المطابِقة لمشاكل المتجر.

المنطق في: `backend/src/analyzer/services-catalog.js`
